# couplages_stables
Ensemble de bouts de code et d'implémentations d'algorithmes autour des b-couplages stables, dans le cadre de mon TIPE. Vous y trouverez notamment:
- Une implémentation de l'algorithme décrit par Fabien Matthieu dans son article 'On Using Matching Theory to Understand P2P Network Design'. Il permet de trouver un b-couplage stable dans un graphe acyclique.
- Une adaptation naïve de la précédente implémentation dans le cas d'un graphe dynamique, dont les arêtes et sommets sont modifiés périodiquement.
- Une implémentation moins naïve.
- Une étude de l'influence de la présence de cycles au sein du graphe de préférence sur la détermination d'un b-couplage stable.
